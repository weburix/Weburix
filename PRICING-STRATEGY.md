# Weburix pricing strategy — V24, July 2026

Internal working note. It is not a public market-comparison or savings claim.

## Positioning

Weburix is positioned as a lean, affordable provider for small companies, self-employed clients and creators. Public amounts are non-binding starting prices. The written scope, correct VAT wording and final quote remain decisive.

V24 uses coherent, memorable price points rather than mathematically awkward percentage reductions. The prices are normal entry prices, not a permanent sale.

## Scope protection

These low entry prices remain sustainable only if each quote defines:

- page and feature limits
- supplied versus written content
- number of feedback rounds
- third-party costs and ad budgets
- delivery assumptions and client response duties
- what is included in recurring support time

Review prices after 3–5 paid clients or whenever actual delivery time repeatedly exceeds the package calculation.

## Promotion rules

The normal public prices are not shown as permanently discounted. Two separate mechanisms exist:

1. `WEBURIX_PROMOTION` controls a temporary crossed-out-price campaign and stays disabled by default. It requires a genuine reference-price history and a valid end date.
2. `WEBURIX_PROMO_CODES` controls codes in the non-binding enquiry configurator. The default `WEBURIX5` code is time-limited, applies only to one-time services, has a minimum subtotal and a maximum discount. Every code remains subject to confirmation in the final written quote.

Never invent a higher comparison price, conceal conditions or leave a temporary campaign running indefinitely. See `PROMOTION-LEGAL-GUIDE.md`.
